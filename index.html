About

The Infinity Web Data Connector allows for reports made in Infinity to be hooked into Tableau. It works by using the Infinity API, so it supports all current API capabilities. The connector has a UI which builds out the API request within Tableau similar to how generator.webtrends.com built out the older Rest API. Once within Tableau, the data can be refreshed automatically, allowing for dashboards/reports to easily integrate with Webtrends data.
